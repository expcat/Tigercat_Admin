## 💡 说明

- 请在开发前端功能时，优先使用 Tigercat UI 组件。
- 如新增需求或缺陷，请补充具体业务场景、期望行为、影响范围和临时规避方案。
- 尽量避免在本项目中编写复杂的自定义样式或交互逻辑来规避组件限制，应优先通过此文档推动上游改进。
