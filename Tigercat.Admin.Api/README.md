# Tigercat.Admin.Api

Tigercat Admin 后端 API 服务，基于 .NET 10 Minimal API。

## 技术栈

- .NET 10
- Minimal API
- Aspire ServiceDefaults
- CORS
- OpenAPI + Scalar UI
